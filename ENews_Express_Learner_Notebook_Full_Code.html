Define Problem Statement and Objectives

Objective:

The design team of the company has researched and created a new landing page that has a new outline & more relevant content compared to the old page. To test the effectiveness of the new landing page in gathering new subscribers, the Data Science team experimented by randomly selecting 100 users and dividing them equally into two groups. The existing landing page was served to the first group (control group) and the new landing page to the second group (treatment group). Data regarding the interaction of users in both groups with the two versions of the landing page was collected. Being a data scientist in E-news Express, you have been asked to explore the data and perform a statistical analysis (at a significance level of 5%) to determine the effectiveness of the new landing page in gathering new subscribers for the news portal by answering the following questions:

1. Do the users spend more time on the new landing page than on the existing landing page?
2. Is the conversion rate (the proportion of users who visit the landing page and get converted) for the new page greater than the conversion rate for the old page?
3. Does the converted status depend on the preferred language?
4. Is the time spent on the new page the same for the different language users?

Step 1: Define the null and alternate hypotheses

null hypothesis: user time is less than or equal to time spent on the new page vs the old page

alternative hypothesis: user time is greater on the new landing page vs the old page

Step 2: Select Appropriate test

two-independent sample t-test - to compare the sample means from 2 related populations when standard deviation are unknown

  • continuous data - yes the usage time is measured on a continuous scale
  • normally distributed population or sample size > 30
  • independent and random populations - as we are tkaing random samples for two different types of users, the two samples are from two independent populations

Step 3: Decide the significance level

a = 0.05, happy medium

3. Is the conversion and preferred language are independent or related?

  • Variable types: Both categorical (conversion: yes/no, language: English/French/Spanish)

  • Groups: 2 categorical variables

  • Test target: association between variables

  • H: Conversion and preferred_language are independent.
  • H: Conversion and preferred_language are associated (not independent).

Using chisquare test of independence because both variables are categorical and we test association in a contingency table.

  • All expected counts 5
  • Categorical data, independent observations

4. Is the time spent on the new page same for the different language users?

  • Variable type: numeric time spent

  • Groups: 3 independent groups (English/French/Spanish)

  • Test target: means are equal

  • H: _English = _French = _Spanish (means equal across languages)
  • H: At least one mean differs ( = 0.05)

Using oneway ANOVA because the outcome is numeric and we compare means across 3+ independent groups.

  • Independence within/between groups
  • Normality of residuals (or large nn)
  • Equal variances: levene(time_English, time_French, time_Spanish)

import pandas as pd
from scipy.stats import f_oneway

# Filter new page (treatment) users
new_page_data = subscription_data[subscription_data['group'] == 'treatment']

# ANOVA: time by language
groups = [new_page_data['time_spent_on_the_page'][new_page_data['language_preferred'] == lang].dropna() 
          for lang in new_page_data['language_preferred'].unique()]
f_stat, p_value = f_oneway(*groups)
print('F-statistic:', f_stat)
print('p-value:', p_value)

if p_value < 0.05:
    print("Reject H0: Time spent differs by language.")
else:
    print("Fail to reject: Time spent same across languages.")

  • No need to optimize page load speed or content separately by language.

Conclusion and Business Recommendations

Q1: New page conversion > old page? Yes (66% vs 42%, ztest p = 0.008 < 0.05) z = 2.65: Moderate difference in proportions (z measures how many standard errors the difference is from 0). Deploy new page.

Q2: Conversion independent of language? Independent ( = 3.94, df=2, p = 0.14 > 0.05) = 3.94: Small deviation from expected counts under independence. No language optimization needed.

Q3: Time spent same across languages on new page? Yes (F = 0.84, df=(2,116), p = 0.43 > 0.05) F = 0.84: Betweengroup variance is tiny compared to withingroup variation. Consistent across languages.

Q4: Time spent same between new vs old page? No (t = -2.47, df=98, p = 0.014 < 0.05) t = -2.47: New page time mean is 2.47 standard errors higher than old page. New page holds attention longer.

Business Recommendation: Roll out new landing page universally. Converts better, holds attention longer, consistent across languages.
